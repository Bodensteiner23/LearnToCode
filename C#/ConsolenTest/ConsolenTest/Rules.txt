Name des Spiels: Wortratespiel

Beschreibung:

Das Wortratespiel ist ein einfaches textbasiertes Spiel, bei dem der Spieler versucht, 
ein zufällig ausgewähltes Wort zu erraten. 

Das Spiel kann folgendermaßen ablaufen:

Done    Das Spiel beginnt, indem ein zufälliges Wort aus einer vordefinierten Liste von Wörtern ausgewählt wird.
Done    Dieses Wort wird dem Spieler nicht angezeigt. 

Done    Der Spieler hat eine begrenzte Anzahl von Versuchen, um das ausgewählte Wort zu erraten. 
Done    Die Anzahl der Versuche wird zu Beginn des Spiels festgelegt.

Der Spieler gibt einen Buchstaben ein und überprüft, ob dieser Buchstabe im ausgewählten Wort vorkommt. 
Wenn ja, wird der Buchstabe an der richtigen Stelle im Wort angezeigt.
Wenn nicht, wird die Anzahl der verbleibenden Versuche reduziert.


Der Spieler kann Buchstaben eingeben, bis er das Wort erfolgreich errät oder keine Versuche mehr übrig sind.

Das Spiel endet, wenn der Spieler das Wort richtig errät oder keine Versuche mehr übrig sind.

Optionale Erweiterungen:

Fügen Sie eine Punktzahl hinzu, um den Spieler für das erfolgreiche Erraten des Wortes
 zu belohnen und die Leistung zu verfolgen.

Ermöglichen Sie dem Spieler die Wahl der Schwierigkeitsstufe, 
indem Sie die Anzahl der Versuche oder die Auswahl der Wörter anpassen.

Implementieren Sie eine Zeitanzeige, um den Spieler unter Druck zu setzen.

Fügen Sie eine Liste mit Hinweisen hinzu, die der Spieler anfordern kann, um Buchstaben im Wort zu enthüllen.
